# 📊 Olist E-Commerce Sales & Financial Dashboard

## Project Overview

This project analyzes Olist E-Commerce data to evaluate financial performance, order operations, payment behavior, and delivery efficiency through an interactive dashboard built using Excel, Power Query, Power Pivot, and DAX.

---

## Objectives

- Analyze Total Revenue
- Monitor Payment Performance
- Evaluate Expected Revenue
- Measure Revenue Reconciliation
- Analyze Order Status Distribution
- Track Delivery Delays
- Forecast Future Revenue Trends

---

## Data Preparation

The following data preparation steps were performed:

- Handling missing values
- Fixing data types
- Creating calculated columns
- Aggregating payment records using Group By
- Merging datasets using Power Query
- Creating a consolidated Final Table

## Key Business Insights

- Delivered orders generated the highest revenue.
- Credit card was the most frequently used payment method.
- Some months experienced higher delivery delays than others.
- Revenue performance remained close to expected targets.
- Revenue trends can be used to forecast future sales growth.
